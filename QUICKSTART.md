# Быстрый старт

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте браузер по адресу `http://localhost:5173`

## Первые шаги

1. **Зарегистрируйтесь** - создайте учетную запись на странице входа
2. **Создайте тикет** - нажмите "Создать тикет" и заполните форму
3. **Просмотрите тикеты** - все ваши тикеты отображаются на главной странице

## Роли пользователей

### Пользователь (user)
- Создание тикетов
- Просмотр своих тикетов
- Добавление комментариев к своим тикетам

### IT Отдел (it)
- Просмотр всех тикетов
- Взятие тикетов в работу
- Изменение статуса тикетов
- Добавление комментариев ко всем тикетам

### Администратор (admin)
- Все функции IT отдела
- Доступ к админ-панели
- Просмотр статистики и пользователей

## Тестовые данные

Для тестирования создайте несколько пользователей с разными ролями:
- Обычный пользователь (роль: user)
- IT специалист (роль: it) 
- Администратор (роль: admin)

**Примечание**: При регистрации все пользователи создаются с ролью "user" по умолчанию. Для создания пользователей с ролями "it" или "admin" можно временно изменить код регистрации или использовать консоль браузера.

## Структура данных

Все данные хранятся в localStorage браузера:
- `users` - список всех пользователей
- `user` - текущий авторизованный пользователь
- `tickets` - все тикеты системы
